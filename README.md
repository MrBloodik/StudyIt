# StudyIt

## Коротко

StudyIt — простое приложение для обучения. Открытый и некоммерческий проект, доступный для общественного использования.

## Статус

- Текущая версия: **только Android**.

## Установка

```bash
npm install
```

## Запуск

```bash
npx expo start
```

## Варианты запуска

- Android эмулятор или устройство (рекомендуется).
- Development build / Expo Dev Client при необходимости.
